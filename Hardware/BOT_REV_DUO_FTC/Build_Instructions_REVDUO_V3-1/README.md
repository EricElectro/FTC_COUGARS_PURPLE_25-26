# Build Order

This document provides a restructured set of build instructions for the REV Duo V3.1 robot, organized by major subsystems. Designed for portability and ease of use, without the need for building an item before it is needed, these instructions allow builders to focus on one subsystem at a time.

This document can be copied per Robot Build to be used as a checklist during assembly.
